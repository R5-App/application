## Getting Started

```bash
npm install
npm start
```

### Run on Android
```bash
npm run android
```

### Run on iOS
```bash
npm run ios
```

### Run on Web
```bash
npm run web
```

## Tech Stack

- React Native 0.74
- Expo 51
- TypeScript
- React Navigation
- React Native Paper (Material Design 3)
- Axios
- Async Storage
